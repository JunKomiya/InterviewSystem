# 面接練習システム (Interview Practice System)

##
コード内に書くコメントや、antigravity内でのメッセージ、タスクやプランなどは基本的に日本語で表記してください

## 📌 プロジェクトの概要
このプロジェクトは、**インターンの面談（面接）練習を自律的に行えるシステム**の開発を目指したものです。
学生が一人でも本番さながらの面接練習を行い、客観的なフィードバックを得て面接力を向上させることをゴールとしています。

## 🎨 ユーザーインターフェース（UI）
* 本システムの画面（UI）は **Streamlit** を使用して構築します。

## 🎯 実現したい機能（ロードマップ）
AIエージェントは、以下の機能を段階的に実装、または連携できるようにコードを設計してください。

### 1. 面接官の音声読み上げ機能（実装中）
* `edge-tts` を使用し、システム内で面接官の質問音声を自然に再生する。
* 外部プレイヤーを開かず、システム内部（バックグラウンド）で完結させる。

### 2. カメラによる表情・視線検知機能（ベース実装完了）
* `MediaPipe (Face Mesh)` を使用し、練習中のユーザーの表情、瞬き、視線の動きをリアルタイムに検知する。
* 面接中に「しっかりカメラ（面接官）を見ているか」「笑顔を作れているか」を判定する基礎データとする。

### 3. タイピング入力および会話のキャッチボール機能（プロトタイプ実装対象）
* ユーザー（学生）からの回答を、Streamlit上の入力フォームによるタイピング入力で受け付ける。
* 初期フェーズでは生成AIの連携部分を「モック（仮実装）」として動かし、あらかじめ用意した次の質問を面接官に喋らせる。
* 将来的には、音声認識（STT）の実装や、生成AIと連携した自動深掘り質問の生成へアップデートする。

### 4. 診断・フィードバック機能（今後実装）
* 面接終了後、「視線の安定度」「笑顔の割合」「回答の適切さ」などを総合的に評価したレポート画面を表示する。

# 🛠️ 開発環境ガイド（AIエージェント・開発者向け共通ルール）

このプロジェクトの開発環境およびAIエージェント（Antigravity等）への指示用ガイドラインです。
Windows環境特有のライブラリ競合（MediaPipeの互換性エラー）を回避するための最重要ルールが記載されています。

## ⚙️ 開発環境の前提条件

### 1. 使用する Python バージョン
* **Microsoft Store版 Python 3.12** を使用（必須）
* システム標準 of Python 3.13 やその他のバージョンは、ライブラリ未対応のため絶対に使用しないこと。

### 2. 実行・管理コマンドの制限
* 実行、パッケージ管理（pip）の際は、`python` ではなく必ず **`python3.12`** を指名して実行すること。
  * ❌ 誤り: `python test-camera.py` / `pip install ...`
  * ⭕ 正解: `python3.12 test-camera.py` / `python3.12 -m pip install ...`

### 🚨 Streamlitの起動コマンドに関する最重要ルール
* Streamlitアプリを起動・実行する際は、必ず以下の形式で **`python3.12` から `streamlit` モジュールを呼び出す形** で実行すること。
  * ❌ 誤り: `streamlit run app.py` （環境がバグる原因になります）
  * ⭕ 正解: **`python3.12 -m streamlit run app.py`**

### 🚨 Windows環境におけるMediaPipe動作（protobufエラー）の最重要ルール
* Windows環境において、MediaPipe 0.10.x をインポートすると、protobufのバージョン違いにより `Failed to parse: node { calculator: "ImagePropertiesCalculator" ... }` というエラーが発生し、目線認識が動作しなくなります。
* これを防ぐため、システムを起動する前に必ず環境変数 `PROTOCOL_BUFFERS_PYTHON_IMPLEMENTATION=python` を設定する必要があります。
* 起動の際は、自分でコマンドを入力するのではなく、プロジェクトフォルダに用意されている **`run.bat`** または **`run.ps1`** を使用して起動してください。これにより必要な環境変数が自動的に適用された状態で Streamlit が立ち上がります。

### 3. 仮想環境（.venv）の禁止
* Windows環境において、VS Codeのターミナルが古い環境キャッシュを掴んで離さないバグが発生したため、**仮想環境（`.venv`）は一切使用しない。**
* パッケージはすべてグローバル（外の世界の Python 3.12）に直接インストールして管理すること。

## 📦 主要ライブラリのバージョン固定

環境破壊（AttributeErrorの再発）を防ぐため、以下の動作確認済みバージョンを維持すること。AIエージェントによる自動アップデートは禁止。

| ライブラリ名 | 固定バージョン | 役割 | 備考 |
| :--- | :--- | :--- | :--- |
| **mediapipe** | `0.10.14` | 顔認識・メッシュ抽出 | Windowsでの動作安定版。これより上はバグあり |
| **streamlit** | 最新（互換性あり） | Web画面（UI）の構築 | 起動時は必ず `python3.12 -m streamlit run` を使用 |
| **opencv-python** | 最新（互換性あり） | カメラ制御・映像表示 | |
| **edge-tts** | 最新（互換性あり） | 音声合成（面接官の声） | |
